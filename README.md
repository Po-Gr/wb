# WB автотест
UI-автотест для онлайн-магазина, написанный на **Java 17** с использованием **Selenium** и **TestNG**.
Проект построен по паттерну **Page Object Model (POM)**. В классах страниц находятся локаторы и действия с элементами.

## Структура проекта
- `src/main/wb/java`
  - `BasketPage`
  - `ProductPage`
- `src/test/wb/java`
  - `OnlineShoppingTest`

## Notes
 - Я решила использовать именно полный POM pattern - действия вынесенны в pages также как и локаторы, чтобы тест оставался простым и читаемым - в нем не было лишней детализации по ожиданиям и проверкам;
 - В тест добавлена проверка что в корзину добавилась именно одна единица товара и не добавились никакие лишние.
